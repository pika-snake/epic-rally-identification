## 第一步：启动确认

### 启动日定义（关键）

**启动日 = 前一天涨幅 ≤5% 且当天涨幅 ≥7% 的交易日**

这是判断真正启动日的核心规则：
- 4月21日扫到的股票，其启动日可能是4月16日（如600961，4个涨停后被扫到）
- 如果用"扫到的那天"当作启动日，会把山顶当启动日来分析
- 扫描脚本需向后多查2个交易日，识别连续涨停序列

### 信号清单

| 信号 | 阈值 | 说明 |
|------|------|------|
| 放量 | 量比>2x，连续2天+ | 资金首次大规模进入 |
| 较大涨幅 | ≥7%（涨停更强） | 确认资金攻击力度 |

### 判断标准

- **放量 + 涨幅≥7%** → 启动信号，可关注
- **放量 + 涨停** → 最强确认信号

### 688677验证

- ✅ 3月27日量比2.9x（前日9,309手→27,281手）+ 涨幅+6.3%（接近7%，启动确认）
- ✅ 3月30日量比2.8x + 涨停板（最强确认）

## 第二步：验证"有行情"（不是普通反弹）

### 信号清单

| 信号 | 阈值 | 说明 |
|------|------|------|
| 融资余额持续增加 | 连续3天+增加 | 不是一次性炒作 |
| 缩量加速 | 股价涨但量比下降 | 主力锁仓 |
| 连续涨停 | 2天+ | 市场合力最强 |

### 关键判断：融资余额 vs 股价背离

**启动前吸筹阶段（最早介入机会）：**
- 股价仍在下跌或横盘
- 融资余额已经拒绝下跌，持续增加
- 这是有组织资金在"悄悄埋伏"，比启动信号早1~4周
- **背离持续时间越长，之后涨幅越大**

**经典底部信号：**
- 股价创阶段新低
- 融资余额创阶段新高
- → 杠杆资金逆势加仓，是有组织资金的提前埋伏

**普通反弹特征：**
- 融资余额跟随股价一起下跌
- 无背离现象

### 688677融资余额吸筹全过程

| 时间 | 股价 | 融资余额 | 阶段 |
|------|------|---------|------|
| 2月5日 | 50元 | 1.52亿 | 下跌中 |
| 3月3日 | 42元(-16%) | 1.72亿(+13%) | 背离出现 |
| 3月23日 | 36元(最低) | 1.75亿(最高) | 背离顶峰 |
| 3月27日 | 39.86元 | — | **放量启动** |
| 4月1日 | 62.79元 | 2.10亿(+24%) | 高开低走加仓 |

**结论**：背离信号比行情启动早了近一个月（2月初~3月27日），且背离持续时间越长、之后涨幅越大。察觉到这个信号，可以在3月27日启动前就做好准备。

### 688677验证

- ✅ 融资余额从1.72亿→3.49亿（持续增加）
- ✅ 4月3日后出现缩量加速
- ✅ 3月30日、4月1日连续涨停

## 第三步：验证"史诗级"（3倍+级别）

### 信号清单

| 信号 | 说明 |
|------|------|
| 催化因素 | 年报超预期/政策利好/行业拐点，三者至少其一 |
| 融资增速 > 股价增速 | 杠杆资金持续涌入，行情未结束 |
| 外资持仓非反向 | 机构重仓才能走更远（外资踏空小票是常态） |
| 估值提升空间 | PE能从高位降到低位（如100x→30x），提供估值锚 |

```

→ Q4单季度业绩爆发
→ 全年业绩反转确认
→ 明年EPS预期切换，估值锚建立
→ 机构敢于重仓锁筹
```

### 688677验证

- ✅ 年报ROE 5.81%→12.88%（V型反转）
- ✅ 4月1日融资余额暴增24%（杠杆资金趁高开加仓）
- ✅ 外资从1.44%降至0.43%（踏空，验证外资非反向指标）
- ✅ 业绩反转后PE有巨大压缩空间

## 核心结论

**实操定位（关键）**：
- 背离信号 = **买入信号**（启动前埋伏，最佳买点）
- 第三步高分 = **持有信心加成**（不是买入依据）
- 真正买点在启动前，不在启动后追涨

**三个最强信号（同时出现时，行情大概率还没走完）：**

1. **融资余额在启动后不减反增**（尤其是高开低走当天大幅增加）
2. **连续涨停 + 缩量加速**（主力锁仓 = 行情没走完）
3. **催化因素存在**（业绩反转/政策利好/行业拐点）

**执行规则：**
- 三信号齐聚 → 重仓持有
- 两信号出现 → 轻仓试错，等待第三信号确认
- 见顶信号任一出现 → 开始分批减仓

## 操作执行脚本

### 扫描器

路径：`scripts/scan_rally_signal.py`（v1.5）

每日扫描涨幅>=7%的股票，基于第二步信号进行行情验证。
v1.6升级：启动日自动识别（修复v1.4的两个bug）
- Bug1: start/end日期颠倒（all_needed[-1]应为start，all_needed[0]应为end）
- Bug2: 找"前一个交易日"用了[i+1]而非[i-1]（方向反了）
- 问题：扫到的日期可能是连续涨停的第N板（如600961在4/21被扫到，已是第4板）
- 解决：从scan_date向前找连续涨停序列的第一板
- 规则：如果当天是涨停板(>=9.5%)，继续向前找，直到找到"前一天<=5% 且 当天>=7%"的日子

v1.7升级：启动日还原打分（针对4板以上股票）
- 问题：4板以上股票在山顶视角评分只有1-2分，但启动日角度可能得3-4分
- 解决：对4板以上股票（board_count >= 4），增加"启动日还原打分"
- 触发条件：board_count >= 4
- 意义：识别"本来就不是黑马" vs "被山顶视角低估的黑马"
- 实测案例：600961（4板山顶，4/21扫到）→ 山顶分2分，还原分2分 → 结论：本来就只有2分，连续涨停靠板块情绪
- 实测案例：003018（4板山顶，4/22扫到）→ 山顶分2分，还原分2分 → 结论：本来就只有2分，不是黑马框架意义上的高分股
- 效果：600961（4/21扫到）→ 正确识别启动日为4/16，第4板

**用法：**
```bash
python scripts/scan_rally_signal.py [日期YYYYMMDD]
# 示例
python scripts/scan_rally_signal.py 20260327  # 扫描启动日
python scripts/scan_rally_signal.py 20260330  # 扫描启动后第2天
python scripts/scan_rally_signal.py            # 默认上一个交易日
```

**性能优化：**
- 融资数据按日期批量查：30个交易日 = 30次API调用（而非900+次）
- 约15~30秒完成全市场扫描

**输出说明：**
- 评分0~4分，代表4个信号各1分
- 背离（启动前）：最早出现，可提前1~4周判断
- 融资持续/缩量加速/连续涨停（启动后）：确认行情延续性
- **评分>=2**：重点关注，等第三步（催化因素）确认

**结论**：688169目前处于"融资客被套"状态。如果庄家在吸筹，最可能在1-3个月后发动。但若5-6月仍融资增、股价不涨，融资客撑不住会砍仓，反而可能砸出坑。建议6月追踪。

### ② 主力量价行为分类（核心判断）

这是区分成功/失败的**最关键维度**：

| 类型 | 行为特征 | 启动后表现 | 代表股票 |
|------|---------|----------|---------|
| **推土机型** | 往上扫货，跌了不买 | 股价持续上涨，连续涨停 | 688268 |
| **接盘机型** | 跌下来才买，不主动推高 | 股价持续低于启动日 | 688169 |

**判断方法**：
- 启动日后若融资增加但股价不涨 → 接盘机型 ⚠️
- 启动日后若融资增加且股价持续上涨 → 推土机型 ✅

**重要例外（板块情绪优先）**：
- 603318水发燃气：背离7天 ✅ + 融资减仓-28% ❌ + 缩量加速 ✅，理论上三信号全错
- 但因为属于"涨停题材板块情绪"，最终6连板+71%
- **结论**：在强势板块情绪面前，个股的融资背离和量比指标都可能被淹没。板块效应是更上位的信号。

## 关键规则：启动日定义

> 扫描时扫到的日期 ≠ 真正的启动日

**定义**：启动日 = 前一天涨幅 **≤5%** 且当天涨幅 **≥7%** 的交易日

**原因**：
- 4月21日扫到600961（+10%）时，它已经是连续第4个涨停
- 真正的启动日是4月16日（第一板），4月21是山顶
- 如果用"扫到的那天"当作启动日，分析会严重失真

**扫描脚本要求**：扫描时向后多查5个交易日，识别连续涨停序列，找到真正的第一板日期

## 附录：脚本说明

**路径**：`scan_rally_signal.py`（与本SKILL.md同目录）

**性能**：
- 融资数据按日期批量查：30个交易日 = 30次API调用（而非数千次）
- 约15~30秒完成全市场扫描

**信号逻辑（第二步量化）**：

| 信号 | 量化标准 | 最高分 |
|------|---------|-------|
| 背离（三维验证） | ①整体斜率背离 + ②后半段背离 + ③逐日背离≥5天，满足≥2项即得分 | 1分 |
| 融资持续增加 | 今日起连续≥3天融资余额增加 | 1分 |
| 缩量加速 | 股价涨 + 量比下降 | 1分 |
| 连续涨停 | 今日+前日涨幅均≥9.5% | 1分 |

**背离三维验证详解**：
- 维度1（整体斜率）：线性回归判断股价整体向下 + 融资整体向上
- 维度2（后半段）：启动日前半段 vs 后半段，后者股价跌但融资增
- 维度3（逐日计数）：逐日背离天数≥5天
- 三选二即确认背离

**信号逻辑（第三步量化 — 催化因素）**：

| 信号 | 量化标准 | 最高分 |
|------|---------|-------|
| 年报催化 | ROE出现V型反转（最新 > 上期×1.2） | 1分 |
| 融资增速 > 股价增速 | 30天区间融资余额增速 > 股价增速 | 1分 |
| 估值合理 | PE(TTM) < 60（低估<30，合理30~60） | 1分 |

**评分规则**：
- 第二步 0~4分 + 第三步 0~3分 = 总分 0~7分
- 总分 >= 5：重点关注，行情大概率还没走完
- 总分 >= 6：三步齐备，史诗级行情确认

**用法**：
```bash
python scan_rally_signal.py 20260327  # 扫描启动日
python scan_rally_signal.py 20260330  # 扫描启动后第2天
python scan_rally_signal.py            # 默认上一个交易日
```
